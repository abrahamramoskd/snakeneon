# 🟢 Snake — Neon Edition

Snake clásico con estética **neon oscura** en un solo archivo Python.

## ✨ Características

- Degradado de color cabeza → cola con efecto trail
- Halo luminoso (glow) en cabeza y frutas
- Partículas de explosión al comer
- Fruta dorada con temporizador visual circular (+50 pts)
- Cuadrícula que cambia de color según el nivel
- Barra de progreso de nivel en el HUD
- Popups de puntos flotantes
- Soporte WASD y flechas

## 🚀 Instalación

```bash
pip install pygame
python snake.py
```

## 🕹️ Controles

| Tecla        | Acción              |
|--------------|---------------------|
| `← ↑ → ↓` o `WASD` | Mover         |
| `P`          | Pausar / reanudar   |
| `R`          | Reiniciar           |
| `ESC`        | Salir               |
